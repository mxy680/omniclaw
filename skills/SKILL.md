# Omniclaw Skills

This directory contains all skill files for the Omniclaw plugin. Each `.SKILL.md` file defines a skill that the agent can use.

Use the `new-integration` skill when adding a new service integration to the MCP server. It covers three auth strategies: Google OAuth, API key/token, and session-cookie auth via Playwright (for platforms without official APIs).
